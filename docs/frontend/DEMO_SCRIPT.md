# M8 Demo Script

## Setup

Use mock LLM and image providers, run migrations and the M7 seed, then start API,
worker, and frontend. Enable demo auth only for the local environment.

## 5-8 minute flow

1. Sign in as the marketing demo user and scan the dashboard.
2. Create a Cyber Legends campaign and start its asynchronous workflow.
3. Follow the job and backend timeline, then inspect safe generated output.
4. Switch to reviewer and approve or request a revision.
5. Generate campaign media and complete media review.
6. Submit human feedback and show acceptance separately from technical success.
7. Switch to manager, inspect prompt experiments and provider comparison.
8. Switch to admin, inspect jobs, alerts, readiness, and n8n integration guidance.

Never enter a provider key, webhook secret, production token, or private document in
the demonstration.
