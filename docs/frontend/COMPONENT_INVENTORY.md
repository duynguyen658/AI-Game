# Component Inventory

## UI primitives

Button, icon button, input, textarea, select, checkbox, label, badge, separator,
tooltip, dropdown menu, dialog, tabs, toast, skeleton, and progress indicator.

## Layout

App shell, permission-aware sidebar, mobile drawer, top bar, breadcrumbs, page header,
action region, skip link, and content container.

## Data and feedback

Data table, status badge, metric strip, definition list, workflow timeline, correlation
ID copy action, empty state, error state, offline indicator, field error, form error
summary, confirmation dialog, and polling status.

## Feature components

Campaign form and detail, job status panel, feedback form, workflow catalog, CSV upload,
document upload, analysis report, document result, media request and review, storyboard
sequence, prompt lifecycle panel, experiment result, provider comparison table,
approval queue, impact charts, jobs table, alerts table, health matrix, and n8n catalog.

Generic primitives do not contain feature fetching, permissions, or domain mutations.
